# html2pdf

WeasyPrint based service for https://github.com/tomazvila/hn2pdf2sn service document convertion part

## Dependencies

WeasyPrint
```bash
pip install WeasyPrint
```

FastAPI
```bash
pip install fastapi
```

uvicorn
```bash
pip install uvicorn
```

## Running the server

```bash
uvicorn main:app --reload
```
